Follow these steps to run the site locally:

1. Install Node.js and npm
Make sure you have Node.js installed. npm comes bundled with Node.js.
To check if it's installed:
node -v
npm -v


If not installed, download and install it from the official site.
2. Clone the Repository
git clone https://github.com/your-username/durhack25.git
cd durhack25


3. Navigate to the Project Directory
cd durhack-next


4. Install Dependencies
npm install


5. Run the Development Server
npm run dev


The site will be available at http://localhost:3000 by default.
